Privacy Policy — VRChat Group Notify

Last updated: April 4, 2026

Overview

VRChat Group Notify is a Chrome extension that helps you create group posts on VRChat. This policy explains what data the extension accesses and how it is handled.

Data Collection

This extension does not collect, transmit, or store any personal data on external servers. All data remains within your browser.

Data Stored Locally

The following data is stored in your browser using the Chrome Storage API:

  • Group profiles (group ID, group name, post templates, event times, visibility settings, role IDs) — stored in chrome.storage.sync
  • Scheduled posts (post content, scheduled date/time, status) — stored in chrome.storage.local
  • Language preference — stored in chrome.storage.sync
  • Theme preference (light/dark mode) — stored in chrome.storage.local

This data is never sent to any third-party server.

Authentication

The extension uses your existing VRChat login session in the browser. It accesses cookies for vrchat.com solely to authenticate API requests. No passwords or credentials are collected, stored, or transmitted by the extension.

Network Communication

The extension communicates only with VRChat servers:

  • https://api.vrchat.cloud/* — to create group posts
  • https://vrchat.com/* — to access authentication cookies

All communication uses HTTPS. No data is sent to any other server.

Permissions

The extension requests the following Chrome permissions:

  • cookies — to read VRChat session cookies for API authentication
  • storage — to save group profiles, scheduled posts, and preferences locally
  • declarativeNetRequest — to attach authentication headers to VRChat API requests
  • alarms — to trigger scheduled posts at the specified time
  • notifications — to notify you when a scheduled post succeeds or fails
  • host_permissions (vrchat.com, api.vrchat.cloud) — required for API communication and cookie access

Third-Party Services

This extension does not use any analytics, tracking, advertising, or third-party services. Your data is never sold, transferred, or shared with any third party for any purpose.

Data Security

All data is stored locally within Chrome's built-in storage mechanism. All network communication with VRChat servers is encrypted via HTTPS. The extension does not transmit data to any server other than VRChat's official API endpoints.

Data Deletion

You can delete all data stored by this extension at any time by:

  • Uninstalling the extension (all local data is removed automatically)
  • Deleting individual profiles or scheduled posts through the extension's settings page

Children's Privacy

This extension is not directed at children under the age of 13.

Disclaimer

This extension is an unofficial, community-made tool and is not affiliated with, endorsed by, or associated with VRChat Inc. "VRChat" is a trademark of VRChat Inc. Use of the VRChat API is at your own risk.

Changes to This Policy

If this policy is updated, the changes will be posted on this page with an updated date.

Contact

If you have questions about this privacy policy, please open an issue on the GitHub repository.

プライバシーポリシー — VRChat Group Notify

最終更新日: 2026年4月4日

概要

VRChat Group Notify は、VRChat のグループ投稿を作成するための Chrome 拡張機能です。本ポリシーでは、この拡張機能がアクセスするデータとその取り扱いについて説明します。

データの収集

本拡張機能は、個人データを外部サーバーに収集・送信・保存することはありません。すべてのデータはお使いのブラウザ内に留まります。

ローカルに保存されるデータ

以下のデータが Chrome Storage API を使用してブラウザ内に保存されます：

  • グループプロファイル（グループ ID、グループ名、投稿テンプレート、イベント時刻、公開範囲設定、ロール ID）— chrome.storage.sync に保存
  • 予約投稿データ（投稿内容、予約日時、ステータス）— chrome.storage.local に保存
  • 言語設定 — chrome.storage.sync に保存
  • テーマ設定（ライト/ダークモード）— chrome.storage.local に保存

これらのデータが第三者のサーバーに送信されることはありません。

認証

本拡張機能は、ブラウザに保存されている VRChat のログインセッションを使用します。API リクエストの認証のためにのみ vrchat.com の Cookie にアクセスします。パスワードや認証情報を本拡張機能が収集・保存・送信することはありません。

ネットワーク通信

本拡張機能は VRChat のサーバーとのみ通信します：

  • https://api.vrchat.cloud/* — グループ投稿の作成
  • https://vrchat.com/* — 認証用 Cookie へのアクセス

すべての通信は HTTPS で暗号化されています。上記以外のサーバーにデータが送信されることはありません。

権限（パーミッション）

本拡張機能は以下の Chrome 権限を使用します：

  • cookies — VRChat のセッション Cookie を読み取り、API 認証に使用
  • storage — グループプロファイル、予約投稿、設定をローカルに保存
  • declarativeNetRequest — VRChat API リクエストに認証ヘッダーを付与
  • alarms — 指定時刻に予約投稿を実行
  • notifications — 予約投稿の成功・失敗を通知
  • host_permissions（vrchat.com、api.vrchat.cloud）— API 通信および Cookie アクセスに必要

第三者サービス

本拡張機能は、アナリティクス、トラッキング、広告、その他の第三者サービスを一切使用しません。ユーザーのデータが第三者に販売・譲渡・共有されることはありません。

データの安全性

すべてのデータは Chrome の組み込みストレージ機能を使用してローカルに保存されます。VRChat サーバーとのすべての通信は HTTPS で暗号化されています。VRChat の公式 API エンドポイント以外のサーバーにデータが送信されることはありません。

データの削除

本拡張機能が保存したデータは、以下の方法でいつでも削除できます：

  • 拡張機能をアンインストールする（すべてのローカルデータが自動的に削除されます）
  • 拡張機能の設定画面から、個別のプロファイルや予約投稿を削除する

児童のプライバシー

本拡張機能は 13 歳未満の児童を対象としたものではありません。
